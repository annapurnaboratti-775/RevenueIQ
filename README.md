Problem Statement : 

Current digital creator platforms lack reliable mechanisms to ensure fairness and transparency in revenue distribution, as engagement metrics such as views, likes, and shares can be easily manipulated through bots or artificial inflation. This leads to unfair earnings, reduced trust among genuine creators, and a lack of clarity for both platform administrators and creators. There is a need for a system that can accurately analyze engagement data, detect anomalies or fraudulent activity, and ensure that revenue is distributed based on authentic performance, while providing clear, transparent insights through an interactive interface.

Proposed Solution :

RevenueIQ is a full-stack revenue intelligence platform that analyzes creator engagement data, detects fake or manipulated activity using anomaly detection, calculates trust scores based on authenticity, and ensures fair and transparent revenue distribution through normalized performance,while providing secure role-based access and interactive dashboards for clear insights.


Tech Stack

→ React.js – Builds dynamic and component-based user interfaces

→ Tailwind CSS – Utility-first framework for responsive styling

→ Axios – Handles API communication between frontend and backend

→ Node.js – Runtime environment for backend development

→ Express.js – Framework for building RESTful APIs

→ MongoDB Atlas – Cloud database for storing application data

→ JWT Authentication – Secures user login and role-based access

→ Recharts – Creates interactive data visualizations and graphs

→ Framer Motion – Adds animations and smooth UI interactions

→ Git and GitHub – Version control and project management

→ Vercel – Frontend deployment platform

→ Render – Backend deployment platform
